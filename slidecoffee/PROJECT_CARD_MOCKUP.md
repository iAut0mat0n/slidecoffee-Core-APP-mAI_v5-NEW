## Responsive Breakpoints

### Desktop (>1280px)
- 4 columns grid
- Large thumbnails (300px width)
- Full metadata visible

- 3 columns grid
- Medium thumbnails (250px width)
- Full metadata visible

### Tablet (768px - 1024px)
- 2 columns grid
- Medium thumbnails (250px width)
- Abbreviated metadata

### Mobile (<768px)
- 1 column grid
- Full-width thumbnails
- Stacked metadata
- Larger touch targets
